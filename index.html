Features

Designed for fast creative experiments.

A focused neural style transfer app for artists, students, and ML learners who want an interactive way to explore how content features and style statistics combine.

⚡

Single-pass transfer

Apply arbitrary styles to content images in one forward pass instead of training a separate model for every style.

🎚️

Adjustable alpha

Control how strongly the style affects the final image, from subtle texture blending to full stylization.

🖼️

Simple upload UI

Use the Flask interface to upload content and style images, generate output, and download the result.

Workflow

From image pair to stylized output.

The app keeps the experience simple while the model handles feature extraction, AdaIN alignment, and image reconstruction behind the scenes.

Upload content

Choose the photo or scene whose structure you want to preserve.

Upload style

Select any artwork or visual reference whose texture and color statistics should guide the output.

Tune alpha

Set the stylization strength between original content and full style transfer.

Generate art

The VGG encoder, AdaIN layer, and decoder produce a stylized image ready to save.

Under the hood

Built on a clean ML pipeline.

The project uses a fixed VGG encoder for feature extraction and a decoder trained to reconstruct images from AdaIN-transformed features.

content → VGG Encoder style → VGG Encoder AdaIN(x, y) = σ(y) · norm(x) + μ(y) features → Decoder output → Stylized Image

PyTorch

Model implementation, tensor operations, encoder, decoder, and training pipeline.

Flask

Lightweight web app for uploading images and serving generated results.

VGG Encoder

Pre-trained normalized VGG network extracts perceptual content and style features.
